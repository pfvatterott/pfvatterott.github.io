My Projects







Best Snow


Best Snow is a website that shows winter sport enthusiasts how the current snow pack compares to previous winter seasons in hundreds of locations across the Western United States.

This project was created by a team of four members. My main role was connecting to and communicating with an API, organizing and interpreting data, and building the Map UI and graphs.

Note Taker


Note Taker is a heroku deployed application that helps an employee keep track of their busy day. With Note Taker they can create new notes, edit them, and delete them when they're finished.

This application uses several technologies including NodeJS, ExpressJS, Javascript, HTML, Bootstrap, and CSS while also using the Unique-Identifier NPM to create Unique IDs for each individual note.


Github









Team Generator


Team Generator is a NodeJS application that helps a manager create a website with their team's information including Name, Employee ID, Email, Office Number, School, and Github profile.

This application is centered around the Inquirer NPM for NodeJS. It prompts the user for data on each employee and then dynamically creates a website for them. The website is then built using Materialize CSS to make sure it is responsive and user friendly.
